Como funciona o 2FA?

Uso normal: você entra com usuário e senha e, em seguida, confirma a entrada com o código de 6 dígitos do aplicativo autenticador.

Plano B: se perder o celular ou ficar sem acesso ao aplicativo, use um código de recuperação no lugar do código normal para entrar uma vez.

Depois do plano B: gere novos códigos de recuperação e recadastre o 2FA para continuar protegido.

Escaneie o QR Code com um autenticador como Google Authenticator, Microsoft Authenticator, Authy ou 1Password.

Desativar 2FA

Use apenas se você realmente quiser retirar a camada extra de proteção.
